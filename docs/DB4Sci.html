DB4Sci is an easy to use web-based self service database deployment application. Users can select a database type and version and DB4Sci creates a stand alone database instance for the user. Each deployed database instance is a separate container with its own unique data volumes. Versions of PostgreSQL, MariaDB, Mongo and Neo4j can be selected. DB4Sci is a fully containerized Flask application.
